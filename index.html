What is AvocadoBot?

AvocadoBot is an IRC bot programmed in C++ targeted for Windows.

What is the goal here?

Nothing. The only goal here is to learn about what things are required to build a robust functional IRC bot.

Isn't this illegal?

Honestly, I'm not sure. I have absolutely ZERO intention of ever adding malicious features such as the ability to spread to other computers using an exploit.

Contributions

All pull requests for bug fixes are welcome. However, should someone want to work on adding features, please contact @mbreedlove.
