What is AvocadoBot?

AvocadoBot is an modular bot programmed in C++ targeted for Windows.

Why?

I have a passion for malware and modern techniques used in the wild. It's rare to find the source code for an information stealing botnet, especially one that is written well and commented. Hopefully AvocadoBot can serve as a guide for anyone interested in how a botnet works under the hood.

What is the goal here?

Creating a skeletal bot that can be easily modified with Proof of Concept code and exploits. Anti-detection techniques will make little appearance to focus on a broad set of controls on the host.

Isn't this illegal?

No, I have zero intention of releasing malicious features. This bot is very simple compared to most criminal malware, it would be extremely easy to detect.
